# Dr. Rahul Gupta – Ophthalmologist Website

Vite + React + Tailwind CSS website for Dr. Rahul Gupta, FIMS Hospital, Sonipat.

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Project Structure

```
src/
├── assets/          # Images and videos (Dr. photo, achievements, patient videos)
├── components/
│   ├── Navbar.jsx          # Sticky responsive navbar with mobile menu
│   ├── Hero.jsx            # Hero section with Dr. photo & general ophthalmology intro
│   ├── About.jsx           # About Dr. Rahul Gupta
│   ├── Services.jsx        # Areas of expertise (6 service cards)
│   ├── Achievements.jsx    # Clinical cases with real before/after images
│   ├── Testimonials.jsx    # Video testimonials
│   ├── Education.jsx       # Education, publications & memberships
│   ├── FAQ.jsx             # Accordion FAQ
│   ├── BookAppointment.jsx # Contact & booking CTA
│   └── Footer.jsx          # Full footer with links & contact
├── App.jsx
├── main.jsx
└── index.css
```
